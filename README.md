# Secret Room Chat

A fully responsive, absolutely private, secure HTML/CSS/JS chat room designed with specific role-based access.

## Features
- **Zero Configuration**: Uses native Vanilla JS and `localStorage` so you don't need a backend for testing.
- **Role-Based Access Control**:
  - `Luna` side for standard messaging, attaching media, and receiving read receipts.
  - `Admin` side for silent monitoring, exporting chats, wiping data, and discreetly replying.
- **Media Attachments**: Drag and Drop support for rich image and video sending.
- **Lightbox**: Click on any media to expand it into a gorgeous fullscreen cinematic view.

## GitHub Pages Deployment
1. Upload this entire folder to your repository.
2. In your repository on GitHub, go to `Settings` > `Pages`.
3. Select the `main` branch (or whichever branch you uploaded the files to).
4. Save, and your chat room is live!

*The `.nojekyll` file ensures GitHub Pages serves all assets reliably.*
